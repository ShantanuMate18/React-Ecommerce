## A. React E-Commerce Frontend :

A simple and responsive e-commerce frontend built using React + Vite.
This project was created as part of a UI Developer Assignment to demonstrate UI design skills, component-based structure, responsiveness, and clean React coding practices.

--------------------------------------------------------------------------------------------------------------------------------

## Features :

1. Navigation Bar

Brand Logo / App Name
Menu links: Men, Women, Kids, Electronics, Home & Living
Search bar
Cart icon with navigation to Cart page

2. Product Listing Page

Displays 8–10 dummy products
Each card includes:
Image
Product Name
Price
Add to Cart button
Responsive grid layout (2–3 columns)

3. Product Details Page

Large product image
Name, price, description
Add to cart functionality

4. Cart Page

Lists all added items
Shows image, name, price
Displays total amount

5. Responsive Design

Grid adjusts based on screen size
Works on desktop, tablet & mobile

## How to Run the Project:

1. Clone the Repository : git clone https://github.com/ShantanuMate18/React-Ecommerce.git

2. Navigate to the project : cd React-Ecommerce

3. Install dependencies : npm install

4. Start development server : npm run dev

5. Your app will start at: http://localhost:5173/
